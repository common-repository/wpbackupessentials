== Description ==

WPBackupEssentials allows you to easily back your entire Wordpress website in a few simple clicks.
You can easily backup both your wordpress website files as well as database information since
WPBackupEssentials conveniently stores each of the above in separate archive files and also be
able to do quick restores - saving you a lot of time if you're a developer! You can
use these archive files with the WPBackupEssentials Migrate script to migrate your entire 
Wordpress website to another server.

You can obtain the <a target="_blank" href="http://www.dovetechnologies.net/wpbackupessentials.html">full edition here</a>
You'll get even more features like automatic backups, cron scheduling, remote backups and more!

== Installation ==

1. Upload `wpbackupessentials` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy our simple to use, awesome plugin!


== Frequently Asked Questions ==

= Why should you backup? =

Did you know that Wordpress is one of the most popular content management systems on the internet?
Because of its popularity, there has been increased attacks and it's always a good idea to have a
backup of your website. Further to this, many hosting providers focus very little on backups
and so its best for you to always have backups in case of critical hardware failure.
